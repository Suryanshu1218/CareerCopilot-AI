# 🚀 CareerCopilot AI

CareerCopilot AI is an AI-powered career assistant that helps users improve their resumes and prepare for their dream jobs using Google Gemini.

## ✨ Features

- 📄 Resume Analysis
- 🎯 Skill Gap Detection
- 📚 Personalized Learning Roadmap
- 💼 Portfolio Project Recommendations
- 🎤 Interview Preparation
- ✨ Resume Improvement Suggestions
- 💼 Job Recommendations
- 📈 Learning Progress Tracker
- 📄 Downloadable PDF Report

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- ReportLab
- PyPDF2
- dotenv

## 📂 Project Structure

```
careercopilot-ai/
│
├── agents/
├── tools/
├── utils/
├── app.py
├── requirements.txt
├── README.md
└── .env
```

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/CareerCopilot-AI.git
```

```bash
cd CareerCopilot-AI
```

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run:

```bash
streamlit run app.py
```
